Desarrolladores Freelance

Empoderamos, formamos a mujeres jóvenes como las mejores Freelance programmer (programador independiente ) y la conectamos con trabajos remoto.

Principales Habilidades

HTML5 HyperText Markup Language (lenguaje de marcas de hipertexto), hace referencia al lenguaje de marcado para la elaboración de páginas web.

CSS3 Hojas de Estilo en Cascada (Cascading Style Sheets) describe cómo se deben mostrar los elementos HTML.

JavaScript lenguaje de programación, se utiliza principalmente del lado del cliente permitiendo crear efectos atractivos y dinámicos en las páginas web.
